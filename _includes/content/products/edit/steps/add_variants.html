• to add Option Types, find the Option Types link, located at the top is a Products dropdown menu
  • open this menu and select the Option Types link from the list.
  • this page will allow you to add, edit and delete option types which may be used by any product you create.
  • continue to add alternative text and variant.
  • Once you’ve added the appropriate option types for your product, you may click the + New Variant link located at the top of the page in the navbar
  • The link will open a new page where you can specify values for the already-setup Option Types that make this variant unique.
  • If you notice that you need to further specify something about this product, you may return the products page, add the needed option types, return the product edit page general tab and correlate the product with the new option type. This will automatically populate the field for the option type you’ve added.
